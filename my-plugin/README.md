# my-plugin

Codex runtime scaffold generated by [plugin-kit-ai](https://github.com/plugin-kit-ai/plugin-kit-ai) `init`.

This lane models repo-local Codex notify integration, separate from the official Codex plugin bundle.

## Status

- Runtime: `go`
- Lane: `codex-runtime`
- Status: `production-ready`, stable default path
- Bootstrap contract: Go `1.22+`

## First Run

```bash
go mod edit -replace=github.com/plugin-kit-ai/plugin-kit-ai/sdk=<absolute-path-to>/sdk/plugin-kit-ai
go test ./...
go build -o bin/my-plugin ./cmd/my-plugin
plugin-kit-ai validate . --platform codex-runtime --strict
./bin/my-plugin notify '{"client":"codex-tui"}'
```
