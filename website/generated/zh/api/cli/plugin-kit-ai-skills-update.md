## plugin-kit-ai skills update

Update external Agent Skills through the npm skills CLI

```
plugin-kit-ai skills update [skills...] [flags]
```

### Options

```
  -g, --global                      update global skills only
  -h, --help                        help for update
  -p, --project                     update project skills only
      --skills-cli-version string   npm skills CLI version to run (default "1.5.5")
  -y, --yes                         skip scope prompt
```
