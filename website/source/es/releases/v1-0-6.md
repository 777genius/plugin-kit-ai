# v1.0.6

Fecha de lanzamiento: `2026-03-30`

## Por qué es importante esta versión

`v1.0.6` es la versión en la que los paquetes de tiempo de ejecución compartidos se convirtieron en la ruta pública clara para los equipos Python y Node en lugar de un flujo de trabajo con soporte medio.

## Qué cambió para los usuarios

- `plugin-kit-ai init` ahora puede desarrollar proyectos Python y Node alrededor de `--runtime-package`
- la ruta compartida `plugin-kit-ai-runtime` ahora es una ruta admitida para las opciones aprobadas Python y Node
- Las plantillas de inicio y los ejemplos públicos se actualizaron para que coincidan con la configuración recomendada actualmente.
- La verificación de la versión y las comprobaciones de sincronización de la versión se reforzaron para que la versión publicada permanezca alineada con la ruta de instalación documentada.

## Qué significa esto en la práctica

- si su equipo quiere la ruta principal que no sea Go, Node/TypeScript más `--runtime-package` ahora es la opción recomendada
- si su equipo es Python-first y permanece en el repositorio local, el paquete de tiempo de ejecución compartido ahora es la ruta admitida preferida
- `doctor` y los mensajes de requisitos de tiempo de ejecución son más claros, por lo que las fallas del entorno son más fáciles de detectar antes de culpar al proyecto.

## Valores predeterminados recomendados después de v1.0.6

- elija Go cuando desee la ruta de producción más sólida y la menor fricción de arranque posterior
- elija Node/TypeScript con `--runtime-package` cuando desee la ruta principal no compatible con Go
- elija Python con `--runtime-package` cuando el proyecto sea repositorio local y el equipo sea intencionalmente Python-primero

## Qué hacer ahora

- Si ya usa Python o Node, vuelva a verificar si `--runtime-package` es ahora el mejor valor predeterminado para su repositorio.
- Si todavía confía en los archivos auxiliares proporcionados como predeterminados a largo plazo, compare esa configuración con la guía actual `plugin-kit-ai-runtime`
- si está empezando de nuevo, los valores predeterminados anteriores son el lugar más sencillo para comenzar
