# v1.0.0

Fecha de lanzamiento: `2026-03-26`

## Qué estableció este comunicado

- el primer contrato público estable para el núcleo CLI
- el conjunto de eventos estable aprobado Claude y Codex
- la primera ruta de integración estable Go SDK
- las reglas de andamio generadas para las rutas estables admitidas

## Estable en esta versión

- CLI: `init`, `validate`, `capabilities`, `install`, `version`
- Claude conjunto de eventos estables: `Stop`, `PreToolUse`, `UserPromptSubmit`
- Codex conjunto de eventos estable: `Notify`
- la ruta de integración raíz Go SDK

## Todavía Beta

- extras opcionales a partir de `plugin-kit-ai init --extras`
- cualquier área futura no promovida explícitamente en el contrato estable

## Llamada de migración

El registro SDK se trasladó de los métodos raíz-Claude a los registradores de plataforma. Si estaba construyendo sobre formas preestables, verifique su cableado SDK antes de tratar `v1.0.0` como línea base de compatibilidad.
