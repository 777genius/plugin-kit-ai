# v1.0.0

Date de sortie : `2026-03-26`

## Ce que cette version a établi

- le premier marché public stable pour le noyau CLI
- l'ensemble d'événements stables Claude et Codex approuvés
- le premier chemin d'intégration stable Go SDK
- les règles d'échafaudage générées pour les chemins stables pris en charge

## Stable dans cette version

- CLI : `init`, `validate`, `capabilities`, `install`, `version`
- Claude ensemble d'événements stables : `Stop`, `PreToolUse`, `UserPromptSubmit`
- Codex ensemble d'événements stables : `Notify`
- le chemin d'intégration racine Go SDK

## Toujours en version bêta

- extras optionnels à partir de `plugin-kit-ai init --extras`
- toute zone future non explicitement promue dans le contrat stable

## Appel de migration

L'enregistrement SDK a été déplacé des méthodes root-Claude vers les bureaux d'enregistrement de la plateforme. Si vous construisiez sur des formes préstables, vérifiez votre câblage SDK avant de traiter `v1.0.0` comme référence de compatibilité.
