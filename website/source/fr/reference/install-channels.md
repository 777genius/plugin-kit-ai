# Installer les chaînes

`plugin-kit-ai` prend en charge plusieurs façons d'installer le CLI lui-même.

| Chaîne | Utiliser quand | Remarques |
| --- | --- | --- |
| Homebrew | Vous souhaitez la configuration utilisateur par défaut | Premier choix recommandé |
| Paquet npm | Votre environnement est déjà centré sur npm | Installe le binaire publié |
| Paquet PyPI | Votre environnement est déjà centré sur Python | Installe le binaire publié |
| Script vérifié | Vous avez besoin d'une méthode d'installation de secours | Configurations extérieures utiles au gestionnaire de packages d'abord |

## Limite importante

Ces packages d'installation ne sont pas documentés en tant que SDKs publics ou APIs d'exécution. Le public API réside sous la section générée [API](/fr/api/).

Lisez [Politique de version et de compatibilité] (/fr/reference/version-and-compatibility) si votre vraie question n'est pas « comment puis-je l'installer ? mais "qu'est-ce qui porte exactement la promesse de compatibilité publique ?".
