# v1.0.0

发布日期：`2026-03-26`

## 此版本确立了什么

- 核心CLI的第一个稳定的公共合约
- 批准的稳定 Claude 和 Codex 事件集
- 第一个稳定的 Go SDK 集成路径
- 为支持的稳定路径生成的脚手架规则

- CLI：`init`、`validate`、`capabilities`、`install`、`version`
- Claude 稳定事件集：`Stop`、`PreToolUse`、`UserPromptSubmit`
- Codex 稳定事件集：`Notify`
- 根 Go SDK 集成路径

## 仍处于测试阶段

- `plugin-kit-ai init --extras` 的可选附加功能
- 任何未明确纳入稳定合约的未来领域
